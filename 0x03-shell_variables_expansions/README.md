This folder has the following scripts
0-alias -> creates an alias to delete all files in current directory
1-hello_you -> displays hello + current linux user
2-path -> adds a directory to PATH
3-paths -> lists all directories in PATH
4-global_variables -> lists all the environment variables
5-local_variables -> list local & global environment, plus functions
6-create_local_variable -> creates a new local variable
7-create_global_variable -> using export creates global variable
8-true_knowledge -> result of $variable and a number
9-divide_and_rule -> result of two divided $variables
10-love_exponent_breath -> result of exponentiated $variables
11-binary_to_decimal -> convertes n from base 2 to base 10
12-combinations -> brace expansion ecluding specified pattern
13-print_float -> prints a number to two decimal places
100-decimal_to_hexadecimal -> converts n from decimal to hexadecimal
101-rot13 -> encodes and decodes using rot13 encryption
102-odd -> prints every other file name
103-water_and_stir -> addition in two different bases, result in another
